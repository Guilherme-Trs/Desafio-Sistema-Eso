# Sistema ESO - Desafio Técnico (Frontend)

Este repositório contém o frontend do desafio técnico Sistema ESO.
O projeto foi desenvolvido com Vite + React + Tailwind.

## Requisitos
- Node 18+
- npm
- Backend do desafio rodando (ver repositório backend) em `http://localhost:3333`
- (Opcional) json-server para mock local: `json-server --watch db.json --port 3000`
